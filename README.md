# Math From Zero to Engineering

A complete mathematics learning journey from Class 1 to Class 12 for software engineers.

## Goals

- Build strong mathematical foundations
- Learn algebra, geometry, calculus, probability
- Connect mathematics with programming
- Prepare for AI/ML, graphics, algorithms, and systems

---

# Roadmap

## Foundation
- Class 1–5: Arithmetic & Number Sense

## Intermediate
- Class 6–8: Algebra & Geometry

## Advanced
- Class 9–10: Core Mathematical Thinking

## Engineering Math
- Class 11–12: Calculus, Probability, Linear Algebra
